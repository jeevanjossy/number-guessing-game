# Number Guessing Game 🎯

A fun Python game where the player guesses a number between 1 and 20. Includes both single-player and two-player modes.

## 🚀 Features
- Single & Two Player Mode
- Scoring System
- Hint System (“Very Close”, “Too Low”, etc.)

## 🛠 Tech Used
- Python
